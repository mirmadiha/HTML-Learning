Project 2: Temperature Conversion

This project is a temperature converter that allows users to convert temperatures between Fahrenheit and Celsius, and vice versa.
Users can input a temperature value in either Fahrenheit or Celsius and select the desired conversion direction. The converter then calculates and displays the converted temperature.
This tool provides a convenient way to quickly convert temperatures for various applications, such as weather forecasting, cooking, or scientific calculations.
